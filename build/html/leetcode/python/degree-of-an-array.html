697. 数组的度¶

https://leetcode-cn.com/problems/degree-of-an-array/

给定一个非空且只包含非负数的整数数组 nums, 数组的度的定义是指数组里任一元素出现频数的最大值。

你的任务是找到与 nums 拥有相同大小的度的最短连续子数组，返回其长度。

示例 1:

输入: [1, 2, 2, 3, 1] 输出: 2 解释: 输入数组的度是2，因为元素1和2的出现频数最大，均为2. 连续子数组里面拥有相同度的有如下所示: [1, 2, 2, 3, 1], [1, 2, 2, 3], [2, 2, 3, 1], [1, 2, 2], [2, 2, 3], [2, 2] 最短连续子数组[2, 2]的长度为2，所以返回2. 示例 2:

输入: [1,2,2,3,1,4,2] 输出: 6 注意:

nums.length 在1到50,000区间范围内。 nums[i] 是一个在0到49,999范围内的整数。

class Solution(object):
    def findShortestSubArray(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        map1 = dict()
        map2 = dict()
        map3 = dict()

        for i, n in enumerate(nums):
            map1[n] = map1.get(n, 0) + 1
            if n not in map2:
                map2[n] = i
            map3[n] = i - map2[n] + 1

        degree = max(map1.values())

        min_len = float('inf')
        for k, v in map1.items():
            if v == degree:
                min_len = min(min_len, map3[k])

        return min_len
